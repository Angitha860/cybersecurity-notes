# MITRE ATT&CK Tactics Overview

## Initial Access

How an attacker first enters an environment.

Examples:

* Phishing Emails
* Stolen Credentials
* Exploiting Vulnerabilities

## Defense Evasion

How an attacker avoids detection.

Examples:

* MFA Disabled
* Antivirus Disabled
* Logs Cleared

## Persistence

How an attacker maintains future access.

Examples:

* Creating backup_admin
* Creating backdoor accounts

## Privilege Escalation

How an attacker gains more permissions.

Examples:

* Adding a user to Domain Admins

## Collection

How an attacker gathers data before theft.

Examples:

* Downloading financial records
* Downloading employee data

## Exfiltration

How an attacker transfers data outside the organization.

Examples:

* Uploading files to external websites

## Key Takeaway

Many attacks follow a sequence:

Initial Access
↓
Defense Evasion
↓
Persistence
↓
Privilege Escalation
↓
Collection
↓
Exfiltration
