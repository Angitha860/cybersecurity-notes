# Multi-Factor Authentication (MFA)

Multi-Factor Authentication (MFA) is a security mechanism that requires more than one factor to verify a user's identity.

## Why MFA is Needed

Passwords can be:

- Stolen
- Guessed
- Reused
- Phished

MFA adds an additional layer of security.

## Authentication Factors

### Something You Know

Examples:

- Password
- PIN
- Security Questions

### Something You Have

Examples:

- Mobile Phone
- OTP Application
- Hardware Token

### Something You Are

Examples:

- Fingerprint
- Face Recognition
- Iris Scan

## Examples

### Password + PIN

Not MFA because both factors are something you know.

### Password + OTP

MFA because it uses two different factors.

### Password + Fingerprint

MFA because it uses two different factors.

## Benefits of MFA

- Reduces account compromise risk
- Protects against stolen passwords
- Improves account security

## What I Learned

- MFA requires multiple authentication factors.
- Passwords alone are not always secure.
- Password + OTP is an example of MFA.
- MFA is commonly used in modern applications and enterprise environments.
