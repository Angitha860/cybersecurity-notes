# Linux Basics

## What is Linux?

Linux is an operating system similar to Windows and macOS.

Many cybersecurity tools and servers run on Linux, which makes Linux knowledge important for cybersecurity professionals.

## Linux Paths

Linux uses:

```text
/
```

Example:

```text
/home/angitha
```

Windows uses:

```text
\
```

Example:

```text
C:\Users\Angitha
```

## pwd

Command:

```bash
pwd
```

Meaning:

Print Working Directory.

Purpose:

Shows the current directory.

Example Output:

```text
/home/angitha
```

## ls

Command:

```bash
ls
```

Purpose:

Lists files and folders in the current directory.

## cd

Command:

```bash
cd Downloads
```

Meaning:

Change Directory.

Purpose:

Move into another folder.

## mkdir

Command:

```bash
mkdir cybersecurity
```

Meaning:

Make Directory.

Purpose:

Creates a new folder.

## touch

Command:

```bash
touch notes.txt
```

Purpose:

Creates an empty file.

## cat

Command:

```bash
cat notes.txt
```

Purpose:

Displays the contents of a file.

## rm

Command:

```bash
rm notes.txt
```

Purpose:

Deletes a file.

## What I Learned

- Linux is a popular operating system used in cybersecurity.
- Linux uses `/` in file paths.
- `pwd` shows the current directory.
- `ls` lists files and folders.
- `cd` changes directories.
- `mkdir` creates folders.
- `touch` creates files.
- `cat` displays file contents.
- `rm` removes files.
