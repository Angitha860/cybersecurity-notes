# Log Correlation

## What is Log Correlation?

Log correlation is the process of combining logs from multiple log sources to identify suspicious activity or security incidents.

A single log may appear normal, but multiple related logs can reveal attacker behavior.

## Example

Firewall Log:
Connection From Russia

VPN Log:
User Login

Active Directory Log:
User Added To Domain Admins

When combined, these logs may indicate a potential account compromise and privilege escalation attempt.

## Why Log Correlation Matters

Log correlation helps analysts:

* Detect complex attacks
* Identify attack patterns
* Reduce false positives
* Improve investigations

## Common Log Sources Used in Correlation

* Windows Logs
* Active Directory Logs
* Firewall Logs
* VPN Logs
* Antivirus Logs
* Application Logs

## Key Takeaway

Multiple logs from different sources often tell a more complete story than a single log alone.
