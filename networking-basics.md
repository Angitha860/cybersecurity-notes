# Networking Basics

## What is an IP Address?

An IP address is a unique logical address assigned to a device on a network so it can communicate with other devices.

Example:
192.168.1.10

## What is DNS?

DNS (Domain Name System) translates domain names such as google.com into IP addresses that computers can understand.

## What is TCP?

TCP (Transmission Control Protocol) is a reliable communication protocol that ensures data arrives correctly and in order.

Examples:
- HTTPS
- SSH
- Email

## What is UDP?

UDP (User Datagram Protocol) is a faster communication protocol that does not guarantee delivery.

Examples:
- DNS
- Video Streaming
- Online Gaming
