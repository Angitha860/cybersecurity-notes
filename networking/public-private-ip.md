# Public vs Private IP

## Private IP

Private IP addresses are used inside a local network.

Examples:
- 192.168.1.10
- 10.0.0.5

Multiple homes can use the same private IP addresses.

## Public IP

Public IP addresses are assigned by the ISP and are visible on the Internet.

Example:
- 49.x.x.x

## DHCP

DHCP automatically assigns IP addresses to devices on a network.

Example:
- Laptop receives 192.168.1.10 from the router.

## NAT

NAT (Network Address Translation) translates private IP addresses into public IP addresses when communicating over the Internet.

Example:

192.168.1.10 → 49.x.x.x

## IPv4

Example:
- 192.168.1.10

Characteristics:
- Uses 32 bits
- Uses dots
- Most commonly used today

## IPv6

Example:
- 2001:db8::1

Characteristics:
- Uses 128 bits
- Uses colons
- Created to solve IPv4 address exhaustion

## What I Learned

- DHCP assigns IP addresses.
- ISP assigns public IP addresses.
- NAT translates private IP addresses into public IP addresses.
- IPv6 was created because IPv4 addresses are limited.
