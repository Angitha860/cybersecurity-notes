# MITRE ATT&CK Basics

## What is MITRE ATT&CK?

MITRE ATT&CK is a framework that describes common attacker behaviors and techniques used during cyber attacks.

It helps security teams understand, detect, and respond to threats.

## Common MITRE Tactics

### Initial Access

How an attacker first enters an environment.

Example:

* Phishing Email
* Exploiting a Vulnerability

### Persistence

How an attacker maintains access.

Example:

* Creating backup_admin
* Creating a backdoor account

### Privilege Escalation

How an attacker gains higher permissions.

Example:

* Adding a user to Domain Admins

### Lateral Movement

How an attacker moves between systems.

Example:

* Accessing HR and Finance servers from a compromised Marketing account

### Collection

Gathering data before theft.

Example:

* Downloading employee records

### Exfiltration

Transferring data outside the organization.

Example:

* Uploading company files to an external website

## Why SOC Analysts Use MITRE

* Understand attacker behavior
* Classify suspicious activities
* Improve threat detection
* Assist in investigations

## Key Takeaway

MITRE ATT&CK provides a common language for describing attacker actions throughout an attack lifecycle.
