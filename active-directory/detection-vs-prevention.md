# Detection vs Prevention

## Prevention

Prevention controls attempt to stop attacks before they succeed.

Examples:

* Multi-Factor Authentication (MFA)
* Strong Password Policies
* Account Lockout Policies
* Firewalls

Example:

Attacker knows password
↓
MFA Challenge
↓
Access Blocked

## Detection

Detection controls identify suspicious or malicious activity.

Examples:

* SIEM Alerts
* Impossible Travel Alerts
* MFA Disabled Alerts
* New Admin Account Alerts

Example:

MFA Disabled
↓
SIEM Generates Alert

## Why Both Are Important

Organizations cannot rely only on prevention.

If prevention fails, detection helps identify the attack so the SOC team can investigate and respond.

## Key Takeaway

Prevention tries to stop attacks.

Detection tries to identify attacks.
