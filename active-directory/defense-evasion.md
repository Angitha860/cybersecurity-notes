# Defense Evasion

## What is Defense Evasion?

Defense Evasion refers to techniques used by attackers to avoid detection by security tools, administrators, and SOC analysts.

The goal is to hide malicious activity and make investigations more difficult.

## Common Examples

### Disable MFA

Attacker gains access
↓
Disables MFA
↓
Future logins become easier

### Clear Logs

Attacker compromises system
↓
Deletes security logs
↓
Makes investigation harder

### Disable Antivirus

Attacker disables antivirus software to prevent malware detection.

## Why SOC Analysts Care

Defense evasion often indicates that an attacker is trying to remain undetected.

Common indicators include:

* MFA Disabled
* Security Logs Cleared
* Antivirus Disabled
* Security Tools Stopped

## Key Takeaway

Defense Evasion is the attacker's attempt to hide activity and avoid detection.
