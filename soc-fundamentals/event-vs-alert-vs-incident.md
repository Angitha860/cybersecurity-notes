# Security Event vs Alert vs Incident

## Security Event

A security event is any observable activity on a system or network.

Examples:

- User login
- File access
- Password change
- Failed login

Events are not necessarily malicious.

## Security Alert

A security alert is a notification generated by a security tool when suspicious activity is detected.

Examples:

- Multiple failed logins
- Login from unusual location
- Detection of a malicious IP

Alerts require investigation.

## Security Incident

A security incident is a confirmed security issue that threatens confidentiality, integrity, or availability.

Examples:

- Brute-force attack
- Malware infection
- Ransomware attack

Incidents require response actions.

## Relationship

Event
↓
Alert
↓
Investigation
↓
Incident
↓
Response

## What I Learned

- Events happen constantly.
- Alerts indicate suspicious activity.
- Incidents are confirmed security issues.
